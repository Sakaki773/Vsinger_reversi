# ADR-0002-拆分核心逻辑并增加调试和打包入口

## 状态

已取代。当前以 [ADR-0006-移除Streamlit并支持棋盘规格和窗口缩放](ADR-0006-移除Streamlit并支持棋盘规格和窗口缩放.md) 为准。

## 背景

项目曾从单文件 Tkinter 脚本拆分为核心规则、调试入口、桌面入口和打包脚本。该阶段的目标是让规则逻辑从 UI 中独立出来，并准备 Windows EXE 交付。

## 当前结论

后续项目明确只保留 Tkinter 桌面入口和 Nuitka 打包链路，浏览器调试链路已移除。当前结构为：

- `core.py`
- `tk_ui.py`
- `tk_run.py`
- `reversi.py`
- `build_nuitka.ps1`
- `pic/`
- `knowledge/`

## 影响

该 ADR 只作为历史记录保留，不再作为当前实现依据。
