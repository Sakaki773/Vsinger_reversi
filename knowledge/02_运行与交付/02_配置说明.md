# 配置说明

## 当前配置状态

项目无需配置文件，无 `.env`、INI、JSON 或本地连接配置。

## 本地状态

以下文件或目录属于本地状态，不应提交：

- `__pycache__/`
- `.venv/`、`venv/`
- `.pytest_cache/`
- `dist/`
- `build/`
- `.nuitka_staging/`
- `*.build/`
- `*.dist/`
- `*.onefile-build/`
- `nuitka-crash-report.xml`
- `.env`
- `*.local`
- `.claude/settings.local.json`

## 未来配置约定

如果后续增加可配置项，例如棋盘主题、AI 难度或窗口尺寸，优先使用脱敏模板文件，并同步更新：

- `README.md`
- `requirements.txt`，如新增依赖
- `knowledge/02_运行与交付/02_配置说明.md`
- 必要时新增 ADR
