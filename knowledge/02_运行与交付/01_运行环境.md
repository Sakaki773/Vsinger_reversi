# 运行环境

## Python

- 使用环境：`C:\Users\admin\.conda\envs\envs2\python.exe`
- 当前验证版本：Python 3.11

## 依赖

桌面运行依赖：

- Tkinter，Python 标准库。
- Pillow，用于缩放棋子素材和淡化背景。

打包依赖：

- Nuitka
- ordered-set
- zstandard

依赖记录在 `requirements.txt`。

## 常用命令

```powershell
& "C:\Users\admin\.conda\envs\envs2\python.exe" .\tk_run.py
powershell -ExecutionPolicy Bypass -File .\build_nuitka.ps1 -LowMemory
```

## Android

- Android 工程位于 `android/`。
- 技术栈：Kotlin + Jetpack Compose。
- Android 不依赖 Python 环境，不读取 `requirements.txt`。
- 需要 Android Studio，或至少 JDK、Android SDK 和 Gradle。

当前工程使用：

- Android Gradle Plugin：9.2.1
- Kotlin Gradle Plugin：2.4.0
- Compose BOM：2026.06.00
- activity-compose：1.13.0

命令行构建：

```powershell
powershell -ExecutionPolicy Bypass -File .\build_android.ps1
```

素材同步：

```powershell
powershell -ExecutionPolicy Bypass -File .\sync_android_assets.ps1
```
