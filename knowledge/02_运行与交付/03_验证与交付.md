## 手工验收

```powershell
& "C:\Users\admin\.conda\envs\envs2\python.exe" .\tk_run.py
```

建议检查：

- 窗口可以放大缩小。
- 棋盘保持正方形、居中，并随窗口自动缩放。
- 窗口不能缩到遮挡棋盘和顶部控制区的尺寸以下。
- 棋盘规格 6x6、8x8、10x10、12x12 切换后能重置并正常落子。
- 言和棋底盘更深，天依棋底盘更浅。
- 单人模式选天依棋时，电脑执言和棋并先走。
- 联机模式只显示骨架状态，不发起真实连接。
- `pic/background.png`、`pic/qizi0.png`、`pic/qizi1.png` 正常加载。

## Android 验证

有 Android Studio 或 JDK/Android SDK/Gradle 后，执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\build_android.ps1
```

默认 APK 输出：

```text
android/app/build/outputs/apk/debug/app-debug.apk
```

修改 `pic/` 后，应先执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\sync_android_assets.ps1
```

当前本机若未配置 `JAVA_HOME`、`ANDROID_HOME` 或 Gradle，则只能验证脚本语法和素材同步，不能完成 APK 构建。
