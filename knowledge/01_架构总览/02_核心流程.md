# 核心流程

## 启动流程

1. 运行 `tk_run.py`，或通过兼容入口 `reversi.py`。
2. 创建 `ReversiApp` 窗口。
3. 初始化 `ReversiGame`，默认棋盘规格为 8x8。
4. 加载淡化背景和棋子素材。
5. 绘制棋盘、棋子和合法落子提示。

## 棋盘规格流程

用户可在开局设置里选择 6x6、8x8、10x10、12x12。切换规格会重置棋局，并保持画布外框稳定。窗口缩放时棋盘保持正方形，格子、棋子和提示点按可用空间自动缩放。

## 单人流程

1. 玩家选择言和棋或天依棋。
2. 电脑自动执另一方。
3. 如果玩家选择天依棋，电脑执言和棋并先走。
4. 玩家落子后，程序检查跳过、结束和电脑回合。

## 双人和联机骨架流程

双人模式不触发电脑落子，双方都由本机玩家点击棋盘。

联机模式当前仍使用本地棋盘交互，但会调用 `NetworkSession.configure_local_piece()` 记录开局选棋，并调用 `NetworkSession.send_move()` 记录预留发送动作。后续接入真实联机时，远端落子应通过 `NetworkSession.poll_remote_move()` 进入。

## 打包流程

1. 执行 `build_nuitka.ps1`。
2. 脚本复制 `core.py`、`tk_ui.py`、`tk_run.py` 和 `pic/` 到临时目录。
3. 运行 `tk_run.py --self-check-imports` 做导入自检。
4. Nuitka 打包 `tk_run.py`，生成 `VsingerReversi.exe`。
